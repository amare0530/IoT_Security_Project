## 溫度分析

| 指標 | 值 | 影響 |
|------|-----|-----|
| **最低溫度** | 9.33°C | 冬季實驗室環境 |
| **最高溫度** | 34.11°C | 異常發熱事件或散熱不足 |
| **平均溫度** | 16.62°C | 常態環境 ~17°C |
| **標準差** | 2.73°C | ±2.73°C 變化 |
| **溫度範圍** | 24.79°C | **關鍵：跨越 25°C 的實測條件** |

**技術意義**:
- Phase 1 測試範圍: 典型 20-25°C
- 真實 Zenodo: 9-34°C （比 Phase 1 寬 10x）
- **結論**: Margin 87 bits 在 Phase 1 溫度範圍內尚未實測驗證

## 電壓分析

| 指標 | 值 | 影響 |
|------|-----|-----|
| **最低電壓** | 3.46987V | 電源劣化 |
| **最高電壓** | 3.67130V | 正常供電 |
| **平均電壓** | 3.64955V | 標稱 3.3V ± 5% |
| **標準差** | 0.01673V | ±16.7 mV 波動 |
| **電壓範圍** | 0.20143V | **關鍵：±10% 變化** |

**技術意義**:
- STM32 規範: 2.7-3.6V 正常運作
- 測定值超過規範上限 (3.67V) → 過壓狀態
- **結論**: 需驗證 Margin 在過壓狀態下的退化

## 交叉分析：溫度 × 電壓

**觀察**:
```
溫度 9.33°C:  電壓 3.64-3.66V (穩定)
溫度 34.11°C: 電壓 3.46-3.53V (應力狀態)
```

**推論**: 
- 高溫下電源供應受限
- PUF 穩定性會同時受到溫度+電壓雙重影響
- **Phase 1 無此複合應力測試**

## 關鍵發現

### 💢 Margin 87 bits 的真實風險

1. **溫度條件外推失敗**
   - 理論 margin: 87 bits (在 20-25°C)
   - 真實環境: 可能在 34°C 時劣化 50% 以上
   - **待驗證**: 實測 Hamming 距離如何隨溫度變化

2. **電壓過壓情況**
   - 測定電壓最高 3.67V，超過 STM32 規範上限
   - **可能原因**: 去耦電容不足或測量誤差
   - **影響**: 可能增加位元翻轉率

3. **群體規模跳躍**
   - Phase 1: 5 設備 FAR = ?
   - 現實場景: 84 設備 FAR = ?
   - **Coupon collector paradox**: FAR 會隨 N 增加而上升

### ✓ 現有有利條件

- 84 設備均勻分布，可進行真正的『母體』評估
- 11 次採樣 × 84 設備 = 可評估長期穩定性
- 溫度和電壓都有記錄，**可做環境相關性析**
